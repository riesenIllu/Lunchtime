# Lunchtime

## Getting Started

## Prerequesites
You need to have the following installed in order to run the application:
- dotnet 5
- node.js
- nuget


## Installation and Running the app

### Just the Frontend
after checking out the repository run in the root folder

```
cd client
npm i 
npm start
```
### With Backend (Coming soon)
```
cd client
npm i
cd ..

```
